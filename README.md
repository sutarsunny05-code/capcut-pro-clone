# CapCut Pro Clone - Professional Video Editor

A powerful web-based video editing application similar to CapCut Pro, built with Next.js, React, and modern web technologies. No VPN required!

## âœ¨ Features

- **Video Editing**
  - Import and preview videos
  - Timeline-based editing
  - Split, duplicate, and delete clips
  - Zoom controls for precise editing

- **Effects & Filters**
  - 10+ professional effects
  - Adjustable intensity
  - Multiple blend modes
  - Real-time preview

- **Text Overlays**
  - Multiple text styles
  - Custom fonts and colors
  - 6+ text animations
  - Adjustable duration

- **Audio Editing**
  - Upload custom audio
  - Built-in music library
  - Volume controls
  - Fade in/out effects
  - Audio effects (Echo, Reverb, Bass Boost)

- **Transitions**
  - 8+ transition effects
  - Customizable duration
  - Multiple easing options
  - Smooth animations

- **Export Options**
  - Multiple format support
  - Quality settings
  - Fast rendering

1. **Import Video:** Click "Choose Video File" or drag and drop
